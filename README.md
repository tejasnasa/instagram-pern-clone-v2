- Backend: TypeScript, Node.js, Express.js, PostgreSQL, Prisma, Jsonwebtoken, Bcrypt, Firebase
- Frontend: TypeScript, React.js, Axios, Tailwind
- Hosting: Vercel, Aiven, Cloudinary
